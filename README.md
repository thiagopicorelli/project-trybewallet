# Trybe Wallet
Trabalho feito na Trybe de uma aplicação de controle de gastos com conversão de moeda, permitindo o usuário a adicionar, editar e remover os gastos da tabela, visualizando o preço total em reais. Também foram feitos os testes da aplicação em Jest, e o controle dos estados no React Redux.
